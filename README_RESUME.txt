RESUME PDF INSTRUCTIONS
=================================================

To make the resume viewer work properly:

1. Obtain your resume PDF
2. Rename it to exactly: Aditya_Patel_Resume.pdf
3. Place the file inside: client/public/assets/

The file should be accessible at:
e:\windows_xp_portfolio\client\public\assets\Aditya_Patel_Resume.pdf

Once placed, it will correctly open when double-clicked
from "My Documents" inside the Windows XP File Explorer!
